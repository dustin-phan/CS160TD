# CS160TD
Group Project for CS160

How To Play:
Enemies are attacking! If they reach the end of the path, you will lose lives! If you hit zero lives, you lose!

Left-click platforms to place or upgrade towers to defend youself.

To move a tower, right-click a tower and the platform you want to move it to!
